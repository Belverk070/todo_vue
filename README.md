# todo_vue
Vue task manager application with possibility of saving data to local storage.
Stack: Vue3, Vue-router, Vuex. 
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
