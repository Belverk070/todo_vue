# todo_vue
Vue task application with possibility of saving and editing data to local storage.
<br>
Stack: 
<br>
<ol>
<li>Vue3</li>
<li>Vue-router</li>
<li>Vuex</li>
</ol>


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
